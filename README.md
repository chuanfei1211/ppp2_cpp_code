
# C++ Programming: Principles and Practice — Code Collection  
《C++ 程序设计原理与实践》示例与习题代码合集

本仓库用于记录本人在学习 **《C++ 程序设计原理与实践（Programming: Principles and Practice Using C++）》** 过程中的全部示例代码、习题解答、以及相关的笔记与实践整理。

仓库采用 **Visual Studio：一个解决方案（Solution）+ 多个项目（Project）** 的结构，每一章对应一个独立项目，便于长期维护、复习、查阅与版本控制。

---

## 📚 仓库结构（按章节组织）



ppp2_cpp_code/
├─ ch03_types/             # 第3章：类型、输入输出等
├─ ch04_computation/       # 第4章：计算、表达式、变量、单位转换
├─ ch05_vectors/           # 第5章：向量、循环、数据结构
├─ ch06_functions/         # 第6章：函数与参数传递
├─ ch07_errors/            # 第7章：错误处理与异常机制
├─ ch08_technicalities/    # 第8章：函数技术细节
├─ ch09_classes/           # 第9章：类与抽象
├─ ch10_input_output/      # 第10章：输入输出流
├─ ch11_custom_io/         # 第11章：自定义输入输出格式
├─ ch12_graphics/          # 第12章：GUI 绘图系统（如 FLTK）
└─ README.md

````

每个章节项目内部包含：

- `examples/` —— 书中示例代码  
- `exercises/` —— 本章习题解答  
- `main.cpp` —— 用于测试、运行本章的入口文件

## 🚀 使用方法

### 1. 克隆仓库
```bash
git clone https://github.com/YOUR_NAME/ppp2_cpp_code.git
````

### 2. 打开解决方案

使用 Visual Studio 打开：

```
ppp2_cpp_code.sln
```

### 3. 运行某章节的代码

1. 在 **解决方案资源管理器** 中找到想要运行的章节项目
2. 右键 → **设为启动项目（Set as Startup Project）**
3. 按 **F5** 或 点击绿色“运行”按钮
